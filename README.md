# Talvium AI - Enhanced Website

This is an enhanced version of the Talvium AI website with modern design elements and improved user experience.

View your app in AI Studio: https://ai.studio/apps/1703247d-e359-422a-b557-b9c5a1c37bf3

## Key Improvements Made

1. **Enhanced Hero Section**:
   - Modern gradient backgrounds with animated elements
   - Improved visual demo with floating elements and enhanced conversation UI
   - Stronger value proposition with clear benefits
   - Social proof with key metrics
   - Enhanced CTA buttons with hover effects

2. **Improved Services Section**:
   - Modern card design with hover effects and shadows
   - Enhanced visual elements with consistent iconography
   - Better organization of deliverables with improved layout
   - Expanded technology stack display with categorized tools

3. **Enhanced Process Section**:
   - Visual timeline with progress indicators and enhanced styling
   - Icon enhancements for each step with clear visual representation
   - Additional value proposition section highlighting key metrics
   - Improved animations and transitions

4. **Modernized Navbar**:
   - Enhanced desktop navigation with hover effects and underlines
   - Improved mobile drawer with better styling and organization
   - Added direct contact information in mobile view
   - Branding improvements with logo and company name

5. **Enhanced Footer**:
   - Modern gradient background with decorative elements
   - Improved contact form with better styling and organization
   - Enhanced contact information display with icons
   - Better visual hierarchy and spacing

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Deployment

The website can be deployed to any static hosting service like Vercel, Netlify, or Google Cloud Run.
